# KawaiiSurvivor
Unity2Dゲームの「KawaiiSurvivor」
Kawaiiランドを襲撃してきたリアリティ怪獣からKawaiiを守り抜け！

ゲームのファイルはGameFileというファイルに入っています。

unity機能のObjectPoolを使った軽量化に力を入れています。
unityのAnimationを使ったUIの表示を意識しました。
コメントをこまめに書くことをコードとしては意識しています。
インスペクターで項目を設定できるようにしてエンジニア以外の人が見ても内容を修正しやすいように意識しています。
